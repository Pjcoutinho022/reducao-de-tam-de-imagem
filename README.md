# ---Programa para redução de tam de imagem para GIT HUB--- 

## A utilização desse código e bem simples!!!

### -Primeiro passo.
  Adicionar sua imagem dentro da pasta desse código.
### -Segundo passo.
  É so substituir o nome da variável "ENTRADA" com o nome da sua "IMAGEM"
### -Ultimo passo.
  Substituir o nome da variável "SAIDA" com o nome que sua nova imagem irá receber.
## Agora só rodar o código que estará feito, a imagem será adicionada na mesma pasta do código.
